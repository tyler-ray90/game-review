# MetaReviews

## MetaReviews allows users to post reviews of their favorite games. Users are able to view, edit, and update any of their personal reviews.  Users can also read a collection of other user reviews. 

# Technologies Used

*Nodes
*Databases
*Mongoose
*Javascript
*Html
*CSS
